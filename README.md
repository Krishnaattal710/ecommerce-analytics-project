## Key Insights

1. The 12-month snapshot shows **65,585 delivered orders** and **10,449,702.42 total revenue**, indicating strong transaction volume.
2. Average Order Value is **159.33**, which provides a baseline for pricing and upsell/cross-sell strategy.
3. Customer satisfaction remains stable with an average review score of **4.12**, suggesting generally positive post-purchase experience.
4. Operational execution is solid with **90.16% on-time delivery**, but the remaining delayed orders still represent a meaningful improvement opportunity.
5. Average delivery time is **12.67 days**, and state-level SLA comparison highlights specific regions where delivery optimization can reduce delays.

## Recommendations

1. Prioritize logistics fixes in the worst-performing states (low on-time %, high delay days) by setting carrier-level SLA targets and weekly monitoring.
2. Increase retention in month 1-2 cohorts using post-purchase campaigns (repeat-order offers, replenishment nudges, category-specific bundles).
3. Focus growth spend on top-performing categories while protecting AOV through bundle pricing and threshold-based free-shipping experiments.
